The custom {{page.title}} element provides automatic scrolling when content is too long to fit within a height-constrained tabset. It should be accompanied by a companion <hx-tab> element.

The element is named after the accessible tabpanel role.

Permitted Parents an <hx-tabcontent> element Permitted Children flow content Events Any of the following:
  • close
  • open
